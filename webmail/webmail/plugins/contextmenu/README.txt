Roundcube Webmail Context Menu
==============================
This plugin adds a context menu to the message list. Menu link are:
mark as read
mark as unread
mark as flagged
mark as unflagged
reply
reply to all
forward
print
delete

Install
=======
* Place this plugin folder into plugins directory of RoundCube
* Add contextmenu to $rcmail_config['plugins'] in your RoundCube config
